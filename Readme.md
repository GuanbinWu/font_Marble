## font_Marble
本项目提供一种个人设计的原创字体，名为 “Marble”。并提供部分设计过程中的工程文件。

### 说明
- **字符集编码**: ISO 10646-1 (Unicode, full)
- **包含字形**：目前包含英文字母、数字和常用英文标点。未来将会逐步更新常用字（希腊字母、中文字符、特殊符号、表情等）。
- **缺失的字形**：通常操作系统会自动回退到某种默认字形，因此不必担心“豆腐块”。本项目曾考虑过混合某种已公开的开源字体作为临时填充，但考虑到混用许可证会为用户造成不便，故放弃。
- **风格**：目前只有常规字体（Regular）。
- **字体效果**：请打开这个[网站](./web_test/test.html)

### 如何使用此字体？
下载./build/Marble-Regular.ttf并安装到本地。或者直接点击[这里](./build/Marble-Regular.ttf)。
